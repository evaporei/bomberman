# bomberman

> :bomb: A slack bot that gets images and post them

## Usage

1. Clone repo
2. Change to the repo directory

### Using lein
3. lein run

### Using docker
3. docker build -t bomberman .
4. docker run -it --rm -p 8088:8080 bomberman
